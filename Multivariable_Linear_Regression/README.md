# Linear Regression is the method of finding the best fit line between scattered and uneven datapoints
<h1> It is highly useful in scenarios like predicting stock prices , checking the prices based on several factors </h1>

# NOTE 
<h2> 1 ) Linear regression can only be visualised when theres two variables / features of data. </h2>
<h2> 2 ) More the data (predictor variables) , more the dimensionality</h2>
<h2> 3 ) For instance , if you want to check linear relationship between area and perimeter , it would be easy to viualise it using matplotlib since there are only two variables (namely Area and Side). Where x1 = Side and y = Area  . </h2>
<h2> 4 ) But if you want to check the price of a car, there are many factors to be considered , like Age, mileage, capacity , etc. Where x1 = age , x2 = mileage ,x3........ and so on and y = price . Since there is a lot of variables (ie more than 2),it will me beyond 2D which is hard to visualise (nearly impossible if exceeding ) . </h2>
<h2> 5 ) The answer or the target value may not be the exact same as of the main data , even if you try to give it the exact thing of the training data , since the model finds the "best - fit" line to fit a set of datapoints. So if you expect the model to give exact answer as the training data, that might not happen </h2>
